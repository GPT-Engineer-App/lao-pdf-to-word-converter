# lao-pdf-to-word-converter

AI for Accurately converting PDF to fully editable Word document with Lao Language Support using google OCR libraries. Then the converted result can be viewed and downloaded.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/lao-pdf-to-word-converter.git
cd lao-pdf-to-word-converter
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
